# OpenClaw AI 网关

OpenClaw AI Gateway — 智能体网关，支持多渠道接入。

## 功能特性

- **多渠道支持** — Telegram、Discord、WhatsApp 等多个通讯平台
- **AI 智能体** — 集成 AI 能力的网关服务
- **Web UI** — 友好的管理界面

## 访问方式

- **Web UI**: `http://<fnOS-IP>:18789`
- **API**: `http://<fnOS-IP>:18789/api`

## 端口

- **18789/tcp** — Web UI 和 API 服务
- **18790/tcp** — 辅助服务端口
