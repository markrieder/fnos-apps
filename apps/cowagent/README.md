# CowAgent for fnOS

AI 超级助手，支持微信/飞书/钉钉/QQ/Web 等多平台接入，基于大语言模型的智能对话。

- **官方网站**: https://docs.cowagent.ai/
- **GitHub**: https://github.com/zhayujie/chatgpt-on-wechat
- **Docker 镜像**: `zhayujie/chatgpt-on-wechat`
- **默认端口**: 9899

## Local Build

```bash
cd apps/cowagent && ./update_cowagent.sh
```
